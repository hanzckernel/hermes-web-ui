# Web Terminal
![Terminal Drawer](assets/screenshots/terminal-drawer.png)

The Web Terminal provides a command-line interface directly in your browser, so you can run local commands and manage environments without leaving your workspace.

## What you can do here
* Open browser-based terminal sessions.
* Run commands in the configured environment.
* Manage multiple sessions and resize terminal panes (where supported).
* View terminal output alongside Chat or file inspection.

## Typical workflow
Open the terminal drawer alongside your active chat to run scripts, start servers, or manage dependencies. If a command fails, you can easily copy the output into your chat session for help or open the files drawer to inspect the related code.

## Key controls
* **Terminal Pane:** Run shell commands and view standard output.
* **Drawer Toggle:** Show or hide the terminal view.
* **Session Management:** Open multiple terminal tabs if supported.

## Notes and limits
* Terminal access may be restricted to required/admin access.
* Terminal commands can read or modify local files and services. Avoid pasting secrets, credentials, or destructive commands into shared contexts.
