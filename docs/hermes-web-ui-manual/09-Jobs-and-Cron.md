# Jobs and Cron
![Jobs Manager](assets/screenshots/jobs-manager.png)

Jobs and Cron enable automated, scheduled background tasks, allowing you to run scripts, use skills, or trigger notifications without manual intervention.

## What you can do here
* Create scheduled jobs with prompts, skills, scripts, or delivery targets.
* Pause, resume, run, edit, or remove jobs.
* Review job history and output.
* Use jobs for monitoring, briefings, reminders, or repeated workflows.

## Typical workflow
To automate a recurring workflow, navigate to the jobs manager and create a new scheduled job. Define the trigger schedule, the required skills or scripts, and any delivery targets. Save the job and let the system run it at the designated time. Periodically review the job history to ensure successful execution and inspect any output generated.

## Key controls
* **Jobs List:** View all scheduled tasks and their current active status.
* **Create Job Modal:** Define new automated workflows, triggers, and targets.
* **Job Actions:** Pause, resume, trigger manually, or delete jobs.
* **History View:** Review the execution logs and output of past runs.

## Notes and limits
* Jobs can execute scripts, call tools, or send messages. Review prompts and delivery targets carefully before enabling automation.
* Prefer smoke-testing a job before letting it run unattended.
